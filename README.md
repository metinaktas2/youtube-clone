🎬 YouTube Clone

A modern and fully responsive YouTube clone built with React. This project replicates the core features of YouTube, allowing users to search, explore, and watch videos in a smooth and user-friendly interface.

🚀 Features
🔍 Search for videos using the search bar
🧭 Explore different video categories from the “Explore” section
🎥 Watch videos on a dedicated detail page
📺 View related video suggestions
💬 Display current comments and channel information
🔴 Watch live videos
🎞️ Browse Shorts and videos on the homepage
📱 Fully responsive design (mobile, tablet, desktop)
🧰 Tech Stack
⚛️ React
📡 Axios
🎨 Tailwind CSS
🧭 React Router DOM
🌐 API

src/

└── App.js
⚙️ Installation

Clone the repository:

git clone https://github.com/your-username/youtube-clone.git

Go to the project directory:

cd youtube-clone

Install dependencies:

npm install
▶️ Running the Project
npm start
